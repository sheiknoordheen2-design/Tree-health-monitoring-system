# 🌱 Tree Health Monitoring System

A smart **IoT-based project** designed to monitor tree health using **DHT11, MQ135, Soil Moisture sensor, and ESP32**. The data is processed and displayed on the **Serial Monitor** and integrated with **Blynk IoT** for remote monitoring.


## 🔹 Features

✅ Measures **Temperature & Humidity** (DHT11)
✅ Monitors **Air Quality** (MQ135)
✅ Tracks **Soil Moisture Level**
✅ Real-time data via **Serial Monitor**
✅ Remote monitoring using **Blynk IoT platform**


## 🔹 Hardware Components

* ESP32 Development Board
* DHT11 Sensor (Temperature & Humidity)
* MQ135 Air Quality Sensor
* Soil Moisture Sensor
* Jumper Wires, Breadboard, Power Supply


## 🔹 Software & Tools

* Arduino IDE
* Blynk IoT Platform
* Serial Monitor for debugging/output
* Libraries: `Blynk`, `WiFi`, `DHT`


## 🔹 Working Principle

1. The ESP32 collects data from sensors (Temperature, Humidity, Air Quality, Soil Moisture).
2. Data is displayed on **Serial Monitor** for analysis.
3. Blynk IoT provides an additional platform for **remote data visualization**.
